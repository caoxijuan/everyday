# 变量的声明和使用

## 1.什么是变量？

变量是用来存储数值的区域。

## 2. 如何声明变量？

格式: 关键字(var let)  变量名 (自定义) = 变量值(赋值);
    例子: var name = “糖醋小排";
    注意：变量名最好用英文，其次是拼音，禁止使用中文，因为  中文一般会存在编码问题，“糖醋小排”用双引号，是因为糖醋小排在js中是字符串类型的数据。
